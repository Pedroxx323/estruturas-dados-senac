# Analise de Complexidade

Esta pasta concentra os exemplos de crescimento assintotico e os materiais de apoio sobre notacao Big O.

## Objetivo

O aluno deve usar esta parte do repositorio para relacionar algoritmo, contagem de operacoes e comportamento do tempo de execucao conforme a entrada aumenta.

## Organizacao

- `exemplos-java`: classes com exemplos executaveis.
- `explicacoes`: textos de apoio com interpretacao teorica.

## Complexidades Abordadas

1. `O(1)` tempo constante.
2. `O(n)` tempo linear.
3. `O(log n)` tempo logaritmico.
4. `O(n log n)` tempo linearitmico.
5. `O(n^2)` tempo quadratico.

## Arquivos Principais

- `exemplos-java/TempoConstante.java`
- `exemplos-java/TempoConstanteEncapsulado.java`
- `exemplos-java/TempoLinear.java`
- `exemplos-java/TempoLogaritmico.java`
- `exemplos-java/TempoLinearitmico.java`
- `exemplos-java/TempoQuadratico.java`

## Como Estudar

1. execute a classe Java do tema;
2. observe o que o algoritmo faz;
3. leia o arquivo explicativo correspondente;
4. compare esse crescimento com os outros exemplos da pasta.
