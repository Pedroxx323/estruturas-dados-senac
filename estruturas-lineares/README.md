# Estruturas Lineares

Esta pasta reune as implementacoes praticas de estruturas lineares trabalhadas na disciplina.

## Objetivo

O foco aqui e mostrar como cada estrutura organiza dados, quais operacoes sao mais comuns e quais diferencas praticas aparecem entre modelos baseados em array e modelos encadeados.

## Organizacao

- `arrays`: estrutura estatica com acesso direto por indice.
- `pilhas`: estrutura LIFO implementada com array.
- `filas`: estrutura FIFO implementada com array circular.
- `listas`: estruturas encadeadas simples e duplamente encadeadas.

## Arquivos Principais

- `arrays/ArrayEstaticoExemplo.java`
- `pilhas/PilhaEstaticaArray.java`
- `filas/FilaEstaticaArrayCircular.java`
- `listas/ListaEncadeadaSimples.java`
- `listas/ListaDuplamenteEncadeada.java`

## Ordem Sugerida

1. array estatico;
2. pilha com array;
3. fila com array circular;
4. lista encadeada simples;
5. lista duplamente encadeada.

## Ponto Didatico

Ao estudar esta pasta, vale sempre comparar:

- custo de insercao;
- custo de remocao;
- facilidade de acesso;
- limite de capacidade;
- comportamento da estrutura na memoria.
